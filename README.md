# SharedBanner

##Installation

Run `npm install` in this folder.

##Building

Edit source files found in `/src` only.

Run `grunt` in this folder to build into the `dist` folder.

##Testing

Open `demo/index.html` in your web browser.

##Usage

For now, just include `<script type="text/javascript" src="https://raw.githubusercontent.com/FirefoxUX/SharedBanner/master/dist/fxux-banner.min.js"></script>` in the `<head>` of any page you'd like to include the banner on. To add links to the banner, please submit a pull request with the `src/config.json` updated and a new build in `/dist`.
